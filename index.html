Client alternatif non officiel

Toute ta fac, au même endroit.

l'ent simplifie l'accès à l'ENT de l'Université de Rennes avec une interface unique pour retrouver les notes, l'emploi du temps ADE, les résultats et les services universitaires importants.

  • Connexion avec les identifiants ENT de l'Université de Rennes
  • Tableau de bord unique pour notes, planning, messagerie et services
  • Application web installable sur mobile et ordinateur
  • Projet open source indépendant, pensé pour une consultation plus lisible

JavaScript est nécessaire pour la connexion complète à l'application. Voir le projet sur GitHub.
